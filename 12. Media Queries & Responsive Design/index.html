Responsive Design with Media Queries

Mobile-First Approach

This layout is designed for small screens first, then scaled up for larger viewports using **@media rules**.

The font size of this text changes based on the viewport width using **viewport units**.

Viewport Units

We're using **vw** (viewport width) and **vh** (viewport height) to size elements relative to the browser window. This box's dimensions and text size are responsive.
