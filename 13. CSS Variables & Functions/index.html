CSS Variables & Functions

Custom Properties & var()

The background and border colors of this card are controlled by **CSS variables** defined at the `:root` level. Changing the variable value in the CSS will update both properties at once.

calc(), min(), max(), clamp()

The width of this box uses **calc()** to combine different units. The font size uses **clamp()** to ensure it's always between a minimum and maximum size, while scaling with the viewport. The margin uses **max()** to choose the larger of two values, and the padding uses **min()** to choose the smaller.

attr() Function

The content you see here is generated by the **attr()** function.
